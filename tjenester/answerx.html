Installer plugin

Søk etter "AnswerX" i WordPress plugin-katalogen og trykk Installer → Aktiver.

Gå til AnswerX → Min plan, lim inn nøkkelen fra MotherX-dashboard og lagre.

Plasser shortcode

Bruk [answerx_search] hvor som helst — side, post, widget eller sidebar.

// Enkel shortcode [answerx_search] // Tilpasset [answerx_search placeholder="Spør oss om hva som helst..." button_text="Søk" class="min-container" button_style="background:#7C3AED;color:#fff;" ]

FAQ

Ofte stilte spørsmål.

Fungerer AnswerX med mitt WordPress-tema?

Ja. AnswerX bruker standard WordPress shortcode-API og arver ditt tema — pluss full stil-kontroll på toppen.

Påvirker det hastigheten på siden?

Nei. Widget-et laster asynkront og cacher resultater. PageSpeed-score forblir uendret.

Kan jeg bruke det på flere sider?

Ja. Shortcoden kan brukes i sider, innlegg, sidebars og widgetiserte områder — så mange ganger du vil.

Trenger jeg MotherX-konto i tillegg?

Ja. Start en gratis prøve på MotherX for å få API-nøkkelen, så limer du den inn i plugin-innstillingene.
